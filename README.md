HOW TO TEST

1. Open the level "Splines_Lvl" and hit Play
2. Walk toward any pool lane - a control panel will appear as you enter the zone
3. Use Start / Stop to control the fish
4. Use Forward / Reverse to change direction - watch the fish flip!
5. Use the slider to adjust current speed - keep in mind accelaration is always positive unless Stop
6. Walk to other lanes to test independent courses
7. Each lane has multiple fish - they space automatically on the spline

Tested in UE 5.6 - no additional plugins required
